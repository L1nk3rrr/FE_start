About CSS Selectors

  1. Tag selector - targets elements based on their HTML tag name (e.g., h1, p1)
    Example: p { color: red; } - Selects all <p> elements and sets their text color to red.

  2. ID selector - targets elements based on their id attribute, written with # before the id (e.g., #main_div)
    Example: #error_message { color: red; } - Selects the element with the ID error_message and sets their text color to red.

  3. Class selector - targets elements based on their class name attribute, written with . before the class name (e.g., .some_class_name)
    Example:.blue_text { color: blue; } - Selects all elements with the class blue_text and sets their text color to blue.

  4. Child combinator - selects child elements regardless of nesting depth, using a space between targets (e.g., div .container selects all elements with class .container inside div elements)
    Example: div .button { margin-top: 10px; } - Selects all elements with the class button that are direct children of <div> elements and adds a top margin of 10 pixels.

  5. Descendant combinator - selects direct child elements, using > between targets (e.g., div > .container selects all elements with class .container that are direct children of div elements)
    Example: body > p { font-size: 16px; } - Selects all <p> elements that are direct descendants of the <body> element and sets their font size to 16 pixels.

  6. Adjacent sibling combinator - targets elements directly following another element, using + (e.g., div+span)
    Example: input + button { margin-left: 10px; } - Selects all <button> elements that immediately follow an <input> element and adds a left margin of 10 pixels.

  7. General sibling combinator - targets elements following another element, even with other elements in between, using ~ (e.g., div~span)
    Example: input + button { margin-left: 10px; } - Selects all <button> elements that follow an <h2>,even if there are other elements in between, and adds a left margin of 10 pixels.

  8. Universal selector - targets all elements, using *
    Examples: * { margin-left: 10px; } Selects all elements on the page and sets a left margin of 10 pixels.

  9. Attribute selector - targets elements based on specific attributes (e.g., [attribute], [attribute="value"], [attribute^="value*"])
    Examples: [title] { cursor: pointer; } - Selects all elements that have a title attribute and makes their cursor a pointer when hovering.
    [href="https://www.example.com"] { color: blue; } - Selects all elements with an href attribute that has the exact value https://www.example.com and sets their text color to blue.
